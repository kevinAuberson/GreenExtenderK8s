# experiments

Evidence behind the thesis results — not required to deploy the system
(see the root [README.md](../README.md) for the deployable components).

- [`benchmark/`](benchmark/README.md) — real-cluster benchmark comparing
  the default scheduler against the carbon-aware extender.
- [`portability/`](portability/README.md) — offline simulation checking
  whether the temporal shifting mechanism generalises to grid zones other
  than Switzerland.

Figures generated from each experiment's `results/` data are included
directly in the thesis report and not duplicated in this repository.
